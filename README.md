# 🎵 CodeAlpha Music Player

A modern and responsive music player built using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Front-End Development Internship**.

## ✨ Features

- ▶️ Play & Pause
- ⏮ Previous / ⏭ Next Song
- 🎵 Dynamic Song Title & Artist
- 📀 Album Cover Display
- 📊 Progress Bar
- 🔊 Volume Control
- 📋 Playlist Support
- 🔄 Autoplay
- 📱 Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
